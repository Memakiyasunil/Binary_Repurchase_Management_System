# Binary Repurchase Management System

A robust MERN stack platform for a Binary MLM business model featuring automated tree placements, e-commerce repurchasing for Business Volume (BV), referral links, and a complete admin dashboard.

## Login Access

The database has been seeded with initial mock data. You can log in using the following credentials:

## How to Run

**Backend:**
\`\`\`bash
cd backend
npm install
node server.js
\`\`\`

**Frontend:**
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

## Features Built So Far:
- **Phase 1:** Project setup & DB Config
- **Phase 2:** Authentication (Login, Registration, Nodemailer OTP Email Verification, JWT)
- **Phase 3:** Binary Tree Logic (Recursive placements, visual tree UI, upline/sponsor routing)
- **Phase 4:** E-commerce Models and Store APIs (Repurchase BV generation)
